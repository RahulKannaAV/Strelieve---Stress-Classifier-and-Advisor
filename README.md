# Strelieve---Stress-Classifier-and-Advisor

## Strelieve classifies your stress type based on the conditions you're in, with features extracted from the text response and gives suggestions to overcome the stress

### To run the Project
**Frontend** - Go to frontend directory. Then run the command **npm run dev**

**Backend** - Go to backend directory. Open the **server.py** file to start the server

**LLM** - Make sure to install docker and run **ollama:latest** image to access LLM functionalities
**LLM Model used** - LLAMA 3:8B
